Justificación de la elección del Sistema Operativo
Luego de analizar distintas opciones orientadas a servidores empresariales, se decidió
utilizar Ubuntu Server 24.04 LTS como sistema operativo para la infraestructura del proyecto
SiGeRU.
La elección se realizó considerando aspectos como estabilidad, compatibilidad tecnológica,
facilidad de administración, seguridad y soporte.
Ubuntu Server ofrece una amplia compatibilidad con las tecnologías requeridas para el
proyecto, entre ellas:
Docker,
PHP,
MySQL,
Git,
servicios SSH,
y herramientas de virtualización.
Además, cuenta con soporte LTS (Long Term Support), lo que garantiza actualizaciones de
seguridad y estabilidad a largo plazo.
Otro aspecto considerado fue la gran cantidad de documentación y soporte comunitario
disponible, lo que facilita la resolución de problemas técnicos y la administración del
sistema.
En comparación con otras alternativas analizadas, como CentOS, Red Hat Enterprise Linux
y SUSE Linux Enterprise, Ubuntu Server presenta una administración más accesible y
flexible para el entorno de desarrollo del proyecto, manteniendo igualmente características
orientadas a entornos empresariales.
También se tuvo en cuenta su facilidad para trabajar en entornos virtualizados y
contenedores Docker, tecnología que será utilizada en etapas posteriores para el
despliegue de aplicaciones y servicios del sistema.
Por estos motivos, el equipo considera que Ubuntu Server 24.04 LTS representa la opción
más adecuada para la infraestructura inicial del proyecto SiGeRU.
